## step:
1.install pacakge

```bash
poetry update
```

2.start app

```bash
f5
```



## build app

```bash
docker-compose up -d
```

if you want build a docker image,you can use dockerfile.
such as

```bash
docker build -t fastweb .
docker run -d -p 8000:8000 -e  DB_HOST="192.168.1.4" -m 30m fastweb
```
